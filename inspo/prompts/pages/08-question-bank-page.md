# Question Bank Page Inspiration Prompt

## Page Purpose

Provide a disciplined content-management interface for creating, filtering, and reusing assessment questions.

## Key Visual Hierarchy

- filter toolbar
- searchable question table
- metadata chips for type, difficulty, and topic
- preview drawer or side panel
- create question action

## Target Aesthetic

Table-first, editorially clean, and built for repeated operational use.

## Important Sections And Components

- filter row
- search field
- question rows with truncated stem preview
- quick preview panel
- create and edit entry points

## Interaction Feel

Fast and organized, with emphasis on taxonomy and reuse.

## Layout Constraints

- prefer dense but readable rows
- keep the preview surface visually tied to the table

## Consistency Rules From `design.md`

- same filter and table language as other admin/examiner pages
- difficulty chips use restrained semantic tones
- avoid introducing new card styles
