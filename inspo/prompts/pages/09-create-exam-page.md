# Create Exam Page Inspiration Prompt

## Page Purpose

Guide exam authoring through a structured, confidence-building layout that prevents missed configuration steps.

## Key Visual Hierarchy

- page title and save state
- form sections for metadata, timing, and instructions
- section and question builder area
- assignment panel
- review and publish actions

## Target Aesthetic

Structured, form-centric, and operationally trustworthy.

## Important Sections And Components

- stepper or segmented layout
- cards for exam details and schedule
- question selection builder
- summary sidebar
- save draft and publish controls

## Interaction Feel

Methodical and controlled, with strong progress awareness.

## Layout Constraints

- long-form page with clear section breaks
- persistent summary or action area on desktop

## Consistency Rules From `design.md`

- use shared form fields and card rhythm
- keep primary and secondary actions consistent
- status notices must be clearly separated from editable form content
