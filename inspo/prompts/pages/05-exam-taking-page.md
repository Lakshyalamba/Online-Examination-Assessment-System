# Exam Taking Page Inspiration Prompt

## Page Purpose

Define the focused runtime interface where students answer timed questions with confidence and low distraction.

## Key Visual Hierarchy

- sticky timer and exam status bar
- main question content
- answer control area
- question navigation panel with status chips
- submit button and autosave indicator

## Target Aesthetic

Serious, quiet, highly readable, and task-focused.

## Important Sections And Components

- persistent top exam bar
- split layout with question area and navigation rail
- answer inputs for objective or text responses
- clear saved, saving, and warning states

## Interaction Feel

Precise and low-friction, with visible progress and minimal decorative noise.

## Layout Constraints

- desktop-first two-column layout
- preserve a clean mobile fallback concept
- timer must remain visually dominant but not alarming until needed

## Consistency Rules From `design.md`

- use amber only for urgency windows
- use `IBM Plex Mono` style timing cues
- white content surface with calm slate framing
- strong focus states for keyboard navigation
