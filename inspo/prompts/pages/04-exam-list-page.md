# Exam List Page Inspiration Prompt

## Page Purpose

Help students scan assigned exams, compare availability windows, and confidently start or continue an attempt.

## Key Visual Hierarchy

- page header and filters
- searchable list or table of exams
- action column with Start, Continue, Locked, or Submitted
- clear schedule and duration metadata

## Target Aesthetic

Structured, table-friendly, and highly legible.

## Important Sections And Components

- filters by status and date
- exam list rows or cards
- schedule metadata
- empty state for no active exams

## Interaction Feel

Fast scanning with low ambiguity around eligibility.

## Layout Constraints

- prioritize row clarity over decorative cards
- actions should be easy to locate at a glance

## Consistency Rules From `design.md`

- use shared table styling
- keep status and action language stable
- strong contrast between active and disabled actions
