# UI Inspiration Library

## Purpose

This folder provides visual direction for the Online Examination Assessment System.

It contains:

- detailed prompt sources for generating consistent UI inspiration
- generated reference images for major pages and shared component patterns
- organization notes for page-level and component-level design alignment

## How To Use These Assets

- read [`design.md`](/Users/yashkumar/Online-Examination-Assessment-System/design.md) first
- use these assets to align hierarchy, layout, density, and tone
- treat them as directional references, not exact implementation requirements
- keep the same product identity across landing, dashboards, exam flows, and admin surfaces

## Alignment Rules

All prompts and generated images in this folder should reinforce:

- light-theme-first academic SaaS styling
- deep navy structural color
- teal action and progress accents
- restrained amber warning states
- calm slate surfaces
- high readability and accessible spacing

## Folder Structure

- `prompts/pages/`: page-specific inspiration prompts
- `prompts/components/`: shared component and shell prompt files
- `generated/pages/`: generated page-level image references
- `generated/components/`: generated component-system references
- `pages/`: page catalog notes
- `components/`: component catalog notes

## Important Note

These are direction assets for planning and implementation alignment. Final UI implementation should still follow the detailed page requirements and consistency rules in [`design.md`](/Users/yashkumar/Online-Examination-Assessment-System/design.md).
